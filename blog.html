Important FAQ

What is the Document Object Model (DOM)?

Document Object Model (DOM) হলো একটি প্রোগ্রামিং ইন্টারফেস, যা ওয়েবপেজের HTML বা XML ডকুমেন্টকে একটি গাছের (tree) মতো গঠন করে। DOM-এর মাধ্যমে ওয়েবপেজের প্রতিটি অংশকে (যেমন টেক্সট, ইমেজ, লিংক ইত্যাদি) আলাদা আলাদা অবজেক্ট হিসেবে ধারণ করা হয় এবং জাভাস্ক্রিপ্ট বা অন্যান্য প্রোগ্রামিং ভাষার মাধ্যমে এই অংশগুলোতে পরিবর্তন আনা যায়। DOM মূলত ব্রাউজারে লোড হওয়া ডকুমেন্টকে ডাইনামিকালি পরিবর্তন, মডিফাই বা ম্যানিপুলেট করার সুবিধা দেয়, যা ব্যবহারকারীর ইন্টারঅ্যাকশনের উপর নির্ভর করে পরিবর্তিত হয়।

How do you select an element from the DOM?

DOM থেকে একটি এলিমেন্ট সিলেক্ট করার জন্য বিভিন্ন মেথড ব্যবহার করা হয়। কিছু জনপ্রিয় মেথড হলো:

  1. 1. getElementById(): Select the element using its ID
  2. 2. getElementsByClassName(): Select the element using its class
  3. 3. getElementsByTagName(): Select the element using its tag
  4. 4. querySelectorAll(): Select the element using its attribute
  5. 5. selectAllChildren(): Select the element using its text
What is event delegation in the context of the DOM, and why is it useful?

Event delegation হলো DOM এর এমন একটি কৌশল যেখানে আপনি কোনো নির্দিষ্ট চাইল্ড এলিমেন্টের পরিবর্তে প্যারেন্ট এলিমেন্টে ইভেন্ট লিসেনার অ্যাটাচ করেন। এটি ইভেন্ট বাবলিং (event bubbling) কনসেপ্টের উপর ভিত্তি করে কাজ করে, যেখানে ইভেন্ট চেইন হয়ে উপরের এলিমেন্টে পৌঁছে।

Event delegation কেন দরকারি:

  1. 1. কম ইভেন্ট লিসেনার ব্যবহার করে মেমরি বাঁচানো যায়।
  2. 2. ডাইনামিক্যালি তৈরি হওয়া এলিমেন্টেও ইভেন্ট কাজ করে।
  3. 3. কোড সহজ ও কার্যকর হয়।
How do you manipulate an element's attributes and styles using the DOM?

DOM (Document Object Model) ব্যবহার করে কোনো এলিমেন্টের অ্যাট্রিবিউট এবং স্টাইল পরিবর্তন করা বেশ সহজ।

নিচে ধাপে ধাপে ব্যাখ্যা করা হলো:

  1. 1.getAttribute(): কোনো নির্দিষ্ট অ্যাট্রিবিউটের মান পাওয়ার জন্য।
  2. 2.setAttribute(): অ্যাট্রিবিউটের মান সেট করার জন্য।
  3. 3. removeAttribute(): কোনো অ্যাট্রিবিউট মুছে ফেলার জন্য।
  4. 4. style.property: CSS স্টাইল সেট বা পরিবর্তনের জন্য।
